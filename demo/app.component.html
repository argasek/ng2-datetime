Angular2 datetime component v1.1.0

Default behaviour (plugins default options)

<datetime [(ngModel)]="date"></datetime>

ngModel: {{getDate(date) | date: "medium"}}

With options  -  (See all datepicker options)    (See all timepicker options)

<datetime [timepicker]="{showMeridian: false, minuteStep: 1}" [datepicker]="{daysOfWeekDisabled: [0,6]}" [(ngModel)]="date2"></datetime>

ngModel: {{getDate(date2) | date: "medium"}}

Using Font Awesome

<datetime [(ngModel)]="date" [timepicker]="{icon: 'fa fa-clock-o'}" [datepicker]="{icon: 'fa fa-calendar'}"></datetime>

ngModel: {{getDate(date) | date: "medium"}}

Disable datepicker

<datetime [datepicker]="false" [(ngModel)]="date3"></datetime>

ngModel: {{getDate(date3) | date: "medium"}}

Disable timepicker

<datetime [timepicker]="false" [(ngModel)]="date3"></datetime>

ngModel: {{getDate(date3) | date: "medium"}}
